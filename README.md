# CC Daily

Claude Code / Codex の更新を毎朝・日本語・信頼度ラベル付きで届ける日刊ダイジェスト。
クラウド Routine が毎朝 `issues/<date>.json` を生成し、GitHub Pages で公開する。

**公開URL:** https://oikaze-juku.github.io/cc-daily/
**更新:** 毎朝 6:00 JST に自動（クラウド Routine）

- 設計図 / 計画書: juku-reservation リポの docs/superpowers 配下
- 表示: GitHub Pages（リポ直下 index.html）
